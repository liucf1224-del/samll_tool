# 小工具集合

这是一个使用PyQt实现的小工具集合项目，包含以下组件：

## 主要功能

### Excel文本替换工具
高效的Excel文件批量文本处理工具：
- 支持批量查找和替换操作
- 多工作表同时处理
- 正则表达式匹配支持
- 操作进度实时反馈

### AI聊天工具
智能对话机器人功能：
- 基于AI大模型的自然语言交互
- 上下文感知对话管理
- 流畅的人机对话体验

### 文本翻译工具
多语言文本翻译助手：
- 支持多种主流语言互译
- 批量文本处理能力
- 翻译历史记录管理


### 圣诞树桌面插件
#### 开发时间:25.12.17
基于PyQt5开发的圣诞树桌面插件，具有以下特性：
- 半透明悬浮窗设计，可自由拖拽定位
- 支持多种视觉特效：飘落星星、动态灯带动画
- 粒子系统实现的精美雪花效果
- 右键菜单提供丰富的自定义选项：透明度调节、置顶开关、特效控制等
- 系统托盘集成，方便快速显示/隐藏和退出
- 支持配置持久化，自动保存用户偏好设置

### 圣诞老人桌面插件
#### 开发时间:25.12.19
新增的圣诞老人主题桌面插件，具有以下特色：
- 基于PyQt5的半透明窗口实现
- 圣诞老人图像展示，带有上下浮动动画效果
- 飘落星星特效，营造节日氛围
- 粒子系统实现的彩色星星飘落效果
- 完整的系统托盘功能支持
- 右键菜单提供个性化设置选项
